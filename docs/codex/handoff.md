# MVP Handoff Contract

The runnable MVP must generate `docs/codex/handoff.md` in the target project.

The handoff must tell the next worker:

- the active `task_id`
- the target `cwd`
- the task text
- the active `epoch`
- the allowed `scope`
- the required `docs/codex` read order
- the requirement to preserve unrelated edits and update audit notes

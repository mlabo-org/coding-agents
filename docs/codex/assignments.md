# MVP Role Assignment Contract

The runnable MVP must generate 14 warm-pool role assignments for each target task:

- Intake
- Repo Mapper
- Requirements
- Planner
- Architect
- Implementer
- Test Runner
- Reviewer
- Risk Guard
- Docs Keeper
- UX
- Dependency
- DevOps
- Auditor

Each generated role entry must include `role`, `status`, `task_id`, `epoch`, `scope`, `assignment`, and `expected_output`.
